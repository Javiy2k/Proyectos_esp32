
# Proyectos ESP32

Esta sección presenta una colección de proyectos enfocados en la programación del **ESP32**, con un énfasis especial en el desarrollo de aplicaciones para **Internet de las Cosas (IoT)**. El objetivo es enseñar cómo utilizar el ESP32 de manera eficiente, desde el manejo de sensores y actuadores hasta la transmisión de datos a través de distintos medios de comunicación. Estos proyectos proporcionarán un conjunto de funcionalidades clave que servirán como base para el desarrollo de soluciones innovadoras y diversas en el ámbito del IoT.


## Diagramas de conexión
Aquí se presentan **diagramas de conexión** claros y detallados para la integración de sensores y actuadores con el ESP32. Cada proyecto incluirá su propio diagrama correspondiente, lo que facilita la replicación de los experimentos.

## Explicación de protocolos de comunicación
El ESP32 es capaz de comunicarse mediante diferentes **protocolos de comunicación**, como **Wi-Fi**, **Bluetooth**, **MQTT** y **HTTP**. Aquí encontrarás explicaciones sobre cómo funcionan estos protocolos, en qué situaciones es mejor utilizar cada uno y cómo implementarlos en tus proyectos para enviar y recibir datos de forma eficiente y segura.
